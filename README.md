🐝 Hive

Hive is a powerful ESP32-based wireless HID (Human Interface Device) controller that combines Bluetooth keyboard emulation with a simple, intuitive web dashboard.

Designed for makers, developers, educators, accessibility users, and IT professionals, Hive enables authorized users to remotely send keyboard inputs, shortcuts, text, and system commands to connected devices from any modern web browser.

The device creates its own Wi-Fi access point and hosts a responsive web interface, removing the need for additional software or apps. From the dashboard, users can trigger:

Individual key presses
Keyboard shortcuts and combinations
Custom text input
URLs and automation sequences
Predefined macro actions

Configuration and settings are stored locally on the ESP32, ensuring persistence across reboots.
